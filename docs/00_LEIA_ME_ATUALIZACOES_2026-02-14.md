# Lembrete Psi — Pacote de Atualizações (.md) — 2026-02-14

Este pacote contém **somente documentação (.md)** para você copiar para a pasta `/docs` do projeto e conseguir iniciar um novo chat amanhã **continuando exatamente de onde paramos**.

## O que está incluído
- Atualizações por passo: **21 → 29.1**
- Hotfixes de build aplicados no caminho (para Vercel / Turbopack)

## Como usar (Windows)
1. Baixe o `.zip` deste pacote.
2. Extraia o conteúdo dentro da pasta do seu projeto, em `docs/`:
   - Se preferir: abra o zip e copie **apenas** os `.md` para `/docs`.
3. Faça commit das mudanças em `/docs` (opcional, recomendado).
